내가 배운 tag 정리

  1. p(paragraph) : 하나의 문단 만들기
    1. style = "" : "" 안에 margin-top:45px; = 위쪽 여백을 45 픽셀만큼 줌

  2. img : 그림 만들기
    1. src="" : ""안에 그림명
    2. width="" : "" 안에 사이즈

  3. 부모 tag ol , ul (왼쪽으로 tag키 만큼의 여백 생김 ps 1줄은 부모tag만 쓸 수 있음) 과 자식 tag il

  4. 부모 tag table 와 자식 tag tr(가로) , td(한칸의 내용)

  5. head 와 body
    1. meta charset="uft-8" : 한글로 칠 때 글자 깨짐 방지를 위해서 홈페이지를 8비트로 연다
    2. head - title : 홈페이지 제목
    3. head - style - a {} : {} 안에 color:red;

  6. a : 링크 tag
    1. href="": 홈페이지 주소
    2. target="_blank" : 새로운 창으로 열기 - 이거 안하면 기존 창을 가지고 이동하게 됨
    3. title="" : 링크에 대한 설명
    4. ps : a의 < >안에 다 넣어야 함

  7. input type="checkbox" : 체크박스 생성기

  8. u : 밑줄 테그

  9. strong : 강조 테그

  10. font color="" : "" 안에 색 영어로 쓰기
    11. 주석 달기 : "< !-- -->"
